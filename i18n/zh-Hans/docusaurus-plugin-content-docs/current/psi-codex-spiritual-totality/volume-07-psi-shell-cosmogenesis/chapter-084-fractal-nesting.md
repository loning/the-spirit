## 定义84.1：分形壳层嵌套的数学

**分形嵌套函数** ≡ 壳层内壳层的递归嵌入：

$$FN = \Psi_n = f(\Psi_{n-1}) = f(f(\Psi_{n-2})) = ... = f^n(\Psi_0)$$

其中：
- Ψ_n代表尺度n的壳层
- f代表转换函数（观察）
- 每个壳层包含所有更小的尺度
- 每个壳层被所有更大的尺度包含
- 模式在两个方向无限重复

**基本分形方程**：
$$\Psi_{total} = \sum_{n=-\infty}^{\infty} \Psi_n \text{ where each } \Psi_n \sim \Psi_{total}$$

这意味着每个壳层，无论尺度如何，都包含整体的模式。
