# 第022章：内在小孩作为坍缩锚点

$$IC = \psi_{原初} \cap \psi_{当下}$$

其中交集代表当前体验中可访问的原初意识。

$$\psi_{成人} = \psi_{孩子} + \sum_{i=1}^{n} P_i$$
